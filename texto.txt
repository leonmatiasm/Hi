COMANDOS DE GIT MÁS USADOS

//este código es para meter nuestro proyecto a un repositorio en github, 
//antes deben estar los archivos ya con commit
git remote add origin https://github.com/leonmatiasm/HolaSoyLEMM.git
git push -u origin master

//recuerda que los commandos basicos son
git init
git add .
git status
git commit -m "tu cometario"

//comandos un poco más complicados
git branch "nombreDeLaRama"   **esto crea una rama en el repositorio, y permite trabajar en cosas distintas y luego juntarlas
git checkout "nombreDeLaRama"   **este codigo cambia del branch actual que es en principio master y nos posiciona en "nombreDeLaRama"
git commit   **aqui hacemos commit pero de la rama (branch) en la que estamos, ej. "nombreDeLaRama"
git merge master  **este crea un solo commit pero con el contenido de dos ramas, en este caso se junta el codigo de master en "nombreDeLaRama"
git rebase master  **este hace casi lo mismo que merge, pero es más limpio y nos da un orden de commit ordenada
